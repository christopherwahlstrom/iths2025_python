# Laboration 1 - API Script

Detta är min lösning på laboration 1. Scriptet pratar med API:et för att hämta token, verifiera den och till slut hämta flaggan automatiskt.

## Installation

För att köra scriptet behöver du installera biblioteket `requests`.

```bash
pip install -r requirements.txt
```

## Körning

Starta scriptet med python:

```bash
python main.py
```

Scriptet skriver ut varje steg i terminalen och avslutar med att visa flaggan.
